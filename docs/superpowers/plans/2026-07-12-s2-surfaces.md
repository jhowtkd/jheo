# S2 — Surfaces Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let a new user complete create → audit → generate → publish using only the UI: honest empties/errors, usable AuditRunner, global Audits nav, and project choosers for scoped surfaces.

**Architecture:** Extend Layout nav to eight items; add `GET /api/audits` for a global recent list; reuse Fixes-style project chooser + `jheo.lastProjectId` for Materials/Generations/Channels; pass `{ maxPages, sources }` in audit `config` (job already reads `maxPages`; discovery gains optional source flags); client URL helper before `createProject`; EmptyState/ErrorState rollout; GenerationComposer disabled-hint.

**Tech Stack:** React, React Router, TanStack Query, Fastify + Prisma, Vitest, i18next. Tokens from SI (do not restyle the design system).

**Spec:** `docs/superpowers/specs/2026-07-12-s2-surfaces-design.md`

**Out of scope:** score.ts / evidence (S3), theme tokens (SI), Dashboard re-craft, executive report redesign, telemetry.

### Task 3: GET `/api/audits` + `listAudits` client

**API** in `audits.ts`:

```ts
app.get('/api/audits', async (req, reply) => {
  const limit = Math.min(Number((req.query as { limit?: string }).limit ?? 50), 100);
  const rows = await prisma.audit.findMany({
    orderBy: { createdAt: 'desc' },
    take: limit,
    include: { project: { select: { id: true, name: true, rootUrl: true } } },
  });
  return rows.map((a) => ({
    id: a.id,
    projectId: a.projectId,
    projectName: a.project.name,
    status: a.status,
    score: a.score,
    startedAt: a.startedAt,
    finishedAt: a.finishedAt,
    createdAt: a.createdAt,
  }));
});
```

Add API test (vitest in `apps/api/test/`) creating 2 audits across projects, asserting order + fields.

**Web:** `listAudits(limit?: number)` via `readJsonOrThrow`.

Update `api.ts`:

```ts
export async function runAudit(
  projectId: string,
  config: Record<string, unknown> = {},
): Promise<Audit> {
  const r = await localeFetch(`${API}/audits`, {
    method: 'POST',
    headers: { 'content-type': 'application/json' },
    body: JSON.stringify({ projectId, config }),
  });
  return readJsonOrThrow(r, 'audits');
}
```
